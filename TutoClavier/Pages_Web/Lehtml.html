LE HTML

HTML signifie :HyperText Markup Language est le code utilisé pour structurer une page web.

Le HTML est un langage de description (dit de marquarge)ou (de balisage).


Les balises

Les balises constituent les éléments de base du HTML.

Les balises sont représentées par <nom de la balise>

Exemples : <b> est la balise ouvrante, <b/> , est la baliste fermante

Certaines balises sont dites orpheline, car elle n'ont pas de balise fermante.

Comme <br> est une balise qui force un retour à la ligne, vous pourriez voir cet annotation <br/>, elle est à éviter car déprécie en HTML.

Il est déconseillé de l'utilisé de nos jours, car elle risque de ne plus être reconnue par les navigateurs dans le futur.
